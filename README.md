<h2>Basic Git Commands</h2>

<p>Use the following commands to work with this repository:</p>

<ul>
  <li><strong>Check Git version</strong>
    <pre><code>git --version</code></pre>
  </li>

  <li><strong>Initialise a new repository</strong>
    <pre><code>git init</code></pre>
  </li>

  <li><strong>Check the status of your working directory</strong>
    <pre><code>git status</code></pre>
  </li>

  <li><strong>Stage all changes</strong>
    <pre><code>git add .</code></pre>
  </li>

  <li><strong>Commit staged changes with a message</strong>
    <pre><code>git commit -m "Your commit message"</code></pre>
  </li>

  <li><strong>Add a remote (GitHub)</strong>
    <pre><code>git remote add origin https://github.com/your-username/your-repo.git</code></pre>
  </li>

  <li><strong>Push to the main branch for the first time</strong>
    <pre><code>git push -u origin main</code></pre>
  </li>

  <li><strong>Pull the latest changes</strong>
    <pre><code>git pull</code></pre>
  </li>

  <li><strong>View commit history (compact)</strong>
    <pre><code>git log --oneline --graph --decorate</code></pre>
  </li>

  <li><strong>Create and switch to a new branch</strong>
    <pre><code>git checkout -b feature/my-feature</code></pre>
  </li>

  <li><strong>Switch back to an existing branch</strong>
    <pre><code>git checkout main</code></pre>
  </li>

  <li><strong>Merge a branch into the current branch</strong>
    <pre><code>git merge feature/my-feature</code></pre>
  </li>
</ul>
